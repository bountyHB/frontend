CSS 선택자 01

1. 전체 선택자(*)

HTML 문사 안의 모든 태그를 선택할 때 사용한다.

안녕하세요잉?

2. 태그 선택자 (태그명)

HTML 문서 안의 같은 태그를 모두 선택할 때 사용한다.

3. 아이디 선택자 (#아이디)

HTML 문서 안에 해당하는 아이디 속성값을 가진 태그만 선택할 때 사용한다.

(중복되는 값을 가질 수 없음)
  1. 아이디 선택자 테스트1
  2. 아이디 선택자 테스트2
  3. 아이디 선택자 테스트3
  4. 아이디 선택자 테스트4

4. 클래스 선택자 (.클래스)

HTML 문서 안에 해당하는 클래스 속성값을 가진 태그들을 선택할 때 사용한다.

(중복되는 값을 가질 수 있음)
(여러개의 속성을 가질 수 있음)
  1. 클래스 선택자 테스트 1
  2. 클래스 선택자 테스트 2
  3. 클래스 선택자 테스트 3
  4. 클래스 선택자 테스트 4
  5. 클래스 선택자 테스트 5
